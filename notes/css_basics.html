In case you're not familiar with CSS and how it affects HTML in the browser, here's a quick crash course.

First you target the HTML element you want to change with a "selector". Target all elements with a bare element name like h1.

Classes can be target by prefixing a period '.' character like .content. IDs are targeted with a hash '#' character: #footer.

Once you have your element targeted, you can manipulate it's style by specifying properties and values for those properties. Common properties include color, background, margin, padding, etc.
