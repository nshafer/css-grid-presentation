We adapt the CSS to the size of the browser through Media Queries. These are instructions to the browser to only apply the enclosed styles if the given conditions(s) are met.

The most common media feature expressions used are print, min-width and max-width. Other useful expressions include min-height and screen.

You can include logical conditions with each expression, such as and and or.

Grab the handle in the bottom right corner of the mock browser to resize the example and see the result.
