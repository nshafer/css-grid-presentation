Then came floats. These were the go-to tool for layouts for most of the last decade. They allow you to stack as many blocks next to each other and they resize. Notice that the containing element collapses on itself.

(Next slide) This is commonly fixed with a "clearfix" that clears both sides. A bit of a hack, but it has been an acceptable solution. Now with a flatter markup, we can more easily use media queries.

(Next slide) By default our .col elements will just fill the full width of the browser, which is usually what we want for mobile devices. So we just need to add the float at larger viewport widths!

This is the basis of the grid system in Bootstrap3, with a few more bells and whistles that allows rudimentary reording and multiple breakpoints.
