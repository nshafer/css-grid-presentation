And here's the same layout with CSS Grid! Take a look at the HTML. Isn't that some nice, clean markup? We don't have any notion of layout clogging up our HTML, just content. All control of the layout is in the CSS.

This covers most of the features of CSS Grid. There are a few new concepts to learn, but overall it's not bad. What's more, it's really easy to look at the CSS and get a good idea of what's going on.

We're going to dive in soon, but just to give you an idea, this is what we're doing. First we set display: grid on the container. Then all child elements become items in the grid, just like Flexbox.

From there we define our rows and columns, then assign the items to their spots in the grid. Finally we set a gap between all of the elements. It's basically Tables all over again, but much, much better!
