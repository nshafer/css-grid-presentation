One of the latest tools for layout is Flexbox. Flexbox places all of it's immediately child elements on a single axis. It includes a lot of nice features over floats such as the ability to grow and shrink elements based on a 'weight'

(Next slide) This shows the first column receiving a weight of 3 while the second column gets 1. This effectively makes the width of the first column 3/4, and the second 1/4. This allows a lot more dynamic markup without having to calculate widths.

(Next slide) Another advantage of Flexbox is the ability to align items vertically (cross-axis). This wasn't really possible with floats without some truly horrendous hacks. Here we align them to the bottom.

(Next slide) Flexbox also allows you to easily change the order of each child element with the order property.
