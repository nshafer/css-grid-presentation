An even better way of placing your Grid Items into Grid Areas is with the grid-template-areas property, which is a special kind of new CSS property.

It's a text representation of the grid that visually represents the grid as you want it.

Start by assigning a named Grid Area to each Grid Item with the grid-area property.

Then place those areas in the grid-template-areas of the container, repeating the area names where you want an area to span across cells.
