Define your columns with grid-template-columns. Every time the grid adds a new row, it will use this "template" to define how the columns are sized.

In this case there are two rows added, each of them with 4 columns, each sized at 1/4 of the total width of the container. You can specify absolute sizes (px, em), percentages and more.

What if we wanted 3 equally sized columns? We need a non-integer percent, such as 33.333333334% or to use calc(100/3) and have the browser calculate it.
