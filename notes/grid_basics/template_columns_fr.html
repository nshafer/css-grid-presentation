Along with CSS Grid we get a new unit fr which specifies a fractional amount of the available space.

So in this case, each column is given 1 / 3rd of the space.

Think of it as the number before the fr is the numerator, and the denominator is all of the fr units added up.

You can also do things like 2fr 1fr 1fr to give the first column 2/4ths, then the final two columns get 1/4th each.
