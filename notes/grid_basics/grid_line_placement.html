One method for placing Grid Items is to specify exactly which line each starts and ends at. Remember that Grid Line is the line between each defined column and/or row.

.header shows how to specify each value individually with separate properties. One each for each column/row and each start/end value.

.menu shows an example of using a shortcut to specify the start and end in one property separated by a '/'. Negative values (.content) can be used to count backward from the end.

.footer has another shortcut, which is to specify how many tracks to span the element across.
